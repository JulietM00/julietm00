## 💻 Retail Database Design Project

## 📌 Objective
- This project demonstrates the design and implementation of a relational database using SQL.

- The database includes multiple related tables (customers, products, orders, and order items) connected through primary and foreign keys.

SQL joins and aggregation queries were used to analyze customer behavior, product performance, and overall revenue.

## 📌 Skills  Demonstrated
- Database design
- SQL joins
- Data aggregation
- Relational data modeling
